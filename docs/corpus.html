Corpus Analysis

Home Corpus

In this project, I compared old English texts from the 19th century with modern American English. I used Voyant Tools to see how word usage changes over time.

Word Cloud

The word cloud shows the most common words in the texts. Words like "people," "like," and "know" appear often, especially in modern texts, which are more casual.

Trends

The trends graph shows how often certain words appear in each text. It helps compare older books with modern writing like blogs and spoken language.

Summary

The results show that older texts use longer sentences and more complex words, while modern texts are simpler and easier to understand.
